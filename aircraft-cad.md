## Overview

This project involves the design and assembly of a small aircraft model using SolidWorks. The goal was to develop multi-part mechanical components and integrate them into a complete system with realistic geometry and functional structure.

The project emphasizes 3D modeling of complex shapes, component integration, and collaborative CAD design.

---

## System Design

The aircraft model consists of several subsystems:

- Main body (fuselage)
- Wings and fuel tanks
- Propeller system
- Landing gear (wheels and supports)
- Tail structures (horizontal and vertical stabilizers)

Each subsystem was designed separately and integrated into a full assembly.

## My Contribution

I was responsible for the design and modeling of:

### Wings
- Designed aerodynamic wing geometry with dihedral angle (~5°)
- Created airfoil-like cross sections using spline sketches
- Generated 3D geometry using loft along guide curves

### Fuel Tank
- Modeled streamlined tank using revolved geometry
- Integrated with wing structure using cut and fit constraints

### Propeller System
- Designed central hub using revolved geometry
- Created multi-section blade profiles across multiple reference planes
- Generated full propeller using loft and circular pattern

## Key Techniques

- Lofted surfaces with guide curves
- Spline-based airfoil modeling
- Reference plane construction
- Revolve and pattern operations
- Multi-part assembly constraints

---

## Challenges

- Constructing aerodynamic wing geometry without prior airfoil knowledge
- Managing 3D guide curves for loft operations
- Ensuring compatibility between curved surfaces and attached components
- Aligning multiple parts within assembly constraints

---

## Key Insights

- Complex geometry requires careful planning of reference planes and guide curves
- Loft-based modeling is sensitive to sketch consistency
- Assembly design requires balancing geometry and connectivity
- Iterative refinement is essential for multi-component systems
